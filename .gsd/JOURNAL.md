## 2026-03-23 — Project Initialized

**What happened:**
- Deep questioning session to understand the app vision
- Researched frameworks: React Native + Expo vs Flutter vs SwiftUI
- Decided on React Native + Expo based on existing JS skills + gesture ecosystem
- Discovered MongoDB Realm deprecation (Sept 2025) — avoiding it
- Finalized: local-only v1 with export/import, cloud sync deferred to v1.1
- Created SPEC.md, ROADMAP.md (5 phases), RESEARCH.md, DECISIONS.md

**Key insight:**
The joystick metaphor is central — each pillar is a physical control the user manipulates. The 4-direction swipe creates a 2×2 matrix (positive/negative × direct/indirect) that maps perfectly to CBT behavioral tracking. The hold-to-target gesture adds depth without adding UI clutter.

**Mood:** Excited — the body-fill visualization concept is genuinely creative.
