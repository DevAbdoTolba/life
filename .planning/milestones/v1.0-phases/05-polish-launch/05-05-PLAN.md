Per D-23: App icon is abstract geometric symbol representing balance/three pillars — minimalist, works on both light and dark OS backgrounds.
Per D-24: Splash screen has centered icon with "Hayat" app name text, dark background #0A0A0F.

Generate the icon programmatically using a Node.js script with the `sharp` library for SVG-to-PNG conversion.

1. Install sharp as dev dependency:
   ```bash
   npm install --save-dev sharp
   ```

2. Create `scripts/generate-icons.js` that:

   Defines the base SVG design: 3 overlapping circles arranged in an equilateral triangle pattern representing the 3 pillars. Colors: Afterlife=#F5A623 (top center), Self=#10B981 (bottom left), Others=#3B82F6 (bottom right). Each circle has opacity 0.85 so overlaps create blend effects. Radius 180px each in a 1024x1024 viewBox.

   Base SVG circles (positions for 1024x1024):
   ```
   circle cx="512" cy="380" r="180" fill="#F5A623" opacity="0.85"   (Afterlife: top center)
   circle cx="380" cy="600" r="180" fill="#10B981" opacity="0.85"   (Self: bottom left)
   circle cx="644" cy="600" r="180" fill="#3B82F6" opacity="0.85"   (Others: bottom right)
   ```

   Generates 5 PNG files:

   a) `assets/icon.png` — 1024x1024, solid dark background rect fill="#0A0A0F" covering full canvas, then the 3 circles on top. NO transparency, NO rounded corners (OS applies rounding).

   b) `assets/splash-icon.png` — 1024x1024, TRANSPARENT background (no rect), just the 3 circles centered. Expo splash config handles centering on the dark background.

   c) `assets/android-icon-foreground.png` — 1024x1024, TRANSPARENT background, circles scaled to fit CENTER 66% safe zone (scale all circle cx/cy/r by 0.66 and offset by 170px to re-center). This ensures the design survives any adaptive icon mask shape.

   d) `assets/android-icon-background.png` — 1024x1024, solid dark background rect fill="#0A0A0F", no design elements.

   e) `assets/android-icon-monochrome.png` — 1024x1024, TRANSPARENT background, same circle layout as splash-icon but all fills set to white (#FFFFFF) for Android 13+ themed icons.

3. Run the script: `node scripts/generate-icons.js`

4. Verify app.json config (NO changes needed — paths already correct):
