**Component 2: `src/components/analytics/ComparisonCards.tsx`** per D-19, D-20, D-21.

Props:
```typescript
interface ComparisonCardsProps {
  period: PeriodType;
  currentLogs: Log[];
  dateRange: DateRange;
}
```

This component:
1. Only renders if `period === 'week' || period === 'month'` (per D-21, Today comparison not meaningful). If `period === 'today' || period === 'custom'`, return `null`.
2. Uses `useEffect` to load previous period logs:
   ```typescript
   const [previousLogs, setPreviousLogs] = useState<Log[]>([]);
   const getLogsByPeriod = useLogStore((s) => s.getLogsByPeriod);

   useEffect(() => {
     if (period !== 'week' && period !== 'month') return;
     const prevRange = getPreviousPeriodDates(period);
     getLogsByPeriod(prevRange.start, prevRange.end).then(setPreviousLogs);
   }, [period]);
   ```
3. Computes `PeriodComparison[]` for each pillar using `useMemo`:
   - For each pillar (1, 2, 3):
     - `currentCount`: `currentLogs.filter(l => l.pillarId === pid).length`
     - `previousCount`: `previousLogs.filter(l => l.pillarId === pid).length`
     - `absoluteDelta`: `currentCount - previousCount`
     - `percentageDelta`: `previousCount > 0 ? Math.round(((currentCount - previousCount) / previousCount) * 100) : (currentCount > 0 ? 100 : 0)`

4. Renders stacked cards (D-19):
   - Section title: "Period Comparison" with `fontFamily: typography.fontFamily.semibold, fontSize: typography.sizes.lg, color: colors.textPrimary, marginBottom: spacing.md`
   - **Current period card** (top): `Card` showing pillar names with current counts
   - **Previous period card** (bottom): `Card` showing pillar names with previous counts
   - **Delta indicators** between/below cards: For each pillar show:
     - Pillar name (from `getPillarById(pid).name`)
     - Delta arrow: up arrow + green (`colors.success` #10B981) if positive delta, down arrow + red (`colors.error` #EF4444) if negative, dash + gray if zero
     - Percentage text: `"+15%"` or `"-8%"` or `"0%"`
     - Absolute text: `"(+3)"` or `"(-2)"`

   Layout for delta row: `View` with `flexDirection: 'row', justifyContent: 'space-between'` containing 3 pillar delta items, each in a `View` with `alignItems: 'center'`.

   Card labels: "This Week" / "Last Week" for week period, "This Month" / "Last Month" for month period.
