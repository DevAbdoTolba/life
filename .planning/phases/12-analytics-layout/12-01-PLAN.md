**getDayLabel function** — copy verbatim from PillarBarChart.tsx (lines 16-25):

```typescript
function getDayLabel(day: string, period: PeriodType): string {
  if (period === 'today') return 'Today';
  if (period === 'week') {
    const date = new Date(day + 'T00:00:00');
    return ['Sun', 'Mon', 'Tue', 'Wed', 'Thu', 'Fri', 'Sat'][date.getDay()];
  }
  const date = new Date(day + 'T00:00:00');
  return String(date.getDate());
}
```

**Data transformation** — copy from TrendLineChart.tsx (lines 29-93) with these differences:
- The `useMemo` builds `lineDataSets` exactly like TrendLineChart: collects unique sorted days, builds `pillarTotals: Record<1|2|3, number[]>` summing across all directions per day per pillar, generates xLabels via `getDayLabel` (from PillarBarChart, NOT TrendLineChart — they are identical but use the getDayLabel defined above).
