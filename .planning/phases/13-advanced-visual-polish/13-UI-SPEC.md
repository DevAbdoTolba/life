### Accent Color Reserved-For List

Accent is per-pillar (not a single global color) and is reserved for exactly these elements:

| Pillar | Positive Accent | Negative Accent | Reserved For |
|--------|----------------|-----------------|--------------|
| Afterlife | `#F5A623` golden | `#6B21A8` deep purple | Joystick border, glow shadow, direction chevron (positive/negative valence), knob flash overlay, background hue tint at max drag |
| Self | `#10B981` emerald | `#EF4444` crimson | Same pattern as above |
| Others | `#3B82F6` sky blue | `#64748B` slate | Same pattern as above |

### Background Hue Shift Contract (VIS-05)

- At rest: `#0A0A0F` (neutral)
- At max drag (dragIntensity = 1): interpolate 10–15% toward active pillar's positive color
  - Afterlife max: `#0F0D09` (warm golden tint — approximately 12% blend)
  - Self max: `#090F0C` (cool green tint — approximately 12% blend)
  - Others max: `#090A0F` (blue tint — approximately 12% blend)
- Transition: `withSpring` on release back to `#0A0A0F`
- Driver: existing `dragIntensity` SharedValue (0→1), `interpolateColor()` on SafeAreaView backgroundColor
- Tint always uses positive pillar color regardless of swipe valence (13-CONTEXT.md D-05)

Source: 13-CONTEXT.md D-04, D-05, D-06, D-07

### Liquid Glow Contract (VIS-03)

Primary approach (Skia):
- Skia Canvas overlays the `outerRing` Animated.View
- Renders soft radial blur ring (`BlurMask` or `Shadow` filter) around the joystick circle
- At rest (dragIntensity = 0): blur radius 8px, opacity 0.2, no distortion
- At max drag (dragIntensity = 1): blur radius 16px, opacity 0.6, ring stretched 4px toward drag direction (water-drop elongation)
- Skia layer uses pillar's positive color (matches existing `shadowColor: pillar.positiveColor`)

Fallback (CSS-style shadows if Skia degrades frame rate below 60fps):
- Existing `glowAnimatedStyle` extended: shadowRadius interpolates 8→16, shadowOpacity interpolates 0.2→0.6
- No stretch distortion in fallback — pure radial glow
- Decision: test on lower-end device; if frame rate drops, use fallback (13-CONTEXT.md D-02, performance is priority)

Source: 13-CONTEXT.md D-01, D-02, D-03

## Interaction States

### Joystick Glow States

| State | Visual |
|-------|--------|
| Rest | Outer ring: border width 1.5px pillar.positiveColor, shadow radius 8px opacity 0.2 |
| Dragging (intensity 0→1) | Shadow radius 8→16px, opacity 0.2→0.6, Skia blur ring intensifies |
| Max drag | Skia ring stretches 4px toward drag direction, shadow at full intensity |
| Release | withSpring snap-back on both knob position and glow to rest state |
| Confirmation flash | Existing behavior unchanged — flash overlay 40% opacity, 150ms in / 200ms out |

### Background Hue States

| State | Visual |
|-------|--------|
| Idle | `#0A0A0F` |
| Active drag (any joystick) | Hue shifts toward dragging joystick's pillar positive color, proportional to dragIntensity |
| Release | withSpring eases back to `#0A0A0F` over 300ms |

### Direction Chevron States

| State | Visual |
|-------|--------|
| Rest | opacity: 0 (invisible) |
| Directional hold active | opacity: 1 on active direction, opacity: 0 on other three |
| Hold release | opacity: 0 on all (withTiming 150ms) |
