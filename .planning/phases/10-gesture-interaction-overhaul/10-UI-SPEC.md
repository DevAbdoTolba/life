## Interaction States

This phase introduces new gesture behavior branches. Each state requires a defined visual contract.

### Joystick Idle State
- Outer ring: `colors.surface` fill, `pillar.positiveColor` border at 1.5px width, base glow `shadowOpacity: 0.2`, `shadowRadius: 8`
- Knob: `colors.surfaceLight` fill, `colors.border` border at 1px
- Direction indicators: 4 dots at opacity 0.25, scale 1.0
- No note mode ring visible

### Plain Swipe State (no hold prior)
- Knob tracks thumb, clamped to MAX_DRAG_DISTANCE (40px)
- Active direction indicator: opacity 1.0, scale 1.4
- `dragIntensity` drives glow: `shadowOpacity` ramps 0.2ŌåÆ0.6, `shadowRadius` ramps 8ŌåÆ16
- On release: confirmation flash (pillar color at 0.4 opacity, 150ms in / 200ms out) + scale pulse (spring to 1.08, snap back)
- NoteEntryModal does NOT open (UX-01 gate ŌĆö no prior center-hold)

### Center Hold Detection Branch (BUG-03)
- Trigger condition: LongPress fires (Ōēź 400ms) AND `Math.sqrt(translateX┬▓ + translateY┬▓) < 15px`
- Visual response: `Haptics.impactAsync(Haptics.ImpactFeedbackStyle.Heavy)` fires immediately on hold-start
- Note mode toggles ON: pulsing glow ring appears on knob at `colors.accent` (#F5A623)
  - Pulse animation: `withRepeat(withSequence(withTiming(1, {duration: 600}), withTiming(0.4, {duration: 600})), -1, true)` on ring opacity
  - Ring dimensions: knob diameter + 8px (64px outer), borderWidth 2px, borderColor `colors.accent`, borderRadius 32px, positioned absolutely centered on knob
- Note mode toggles OFF (second center-hold): glow ring fades out (`withTiming(0, {duration: 200})`) and disappears
- No RadialMenu renders for center hold

### Directional Hold Detection Branch (BUG-04)
- Trigger condition: LongPress fires (Ōēź 400ms) AND `Math.sqrt(translateX┬▓ + translateY┬▓) >= 15px`
- Visual response: `Haptics.impactAsync(Haptics.ImpactFeedbackStyle.Medium)` fires on hold-start
- RadialMenu appears: up to 3 active target bubbles fan at 30-degree fixed intervals
  - Fan center angle = swipe direction angle (0┬░=right, 90┬░=up, 180┬░=left, 270┬░=down)
  - With 1 target: single bubble at center angle
  - With 2 targets: bubbles at center┬▒15┬░ (30┬░ total span)
  - With 3 targets: bubbles at center-30┬░, center, center+30┬░ (60┬░ total span)
  - Bubble radius from joystick center: 80px (RADIAL_MENU_RADIUS ŌĆö pre-existing)
  - Bubble size: 44px diameter (RADIAL_BUBBLE_SIZE ŌĆö pre-existing)
  - Bubble fill: `pillar.positiveColor`, white text at 10px / Inter_500Medium
  - Appear animation: staggered spring, 30ms delay per bubble index
  - Hover state (thumb within 30px hit radius): scale 1.2, opacity 1.0; idle: scale 1.0, opacity 0.8
- Cancel escape: if knob returns to center (< 15px) before release, dismiss RadialMenu, no log created
- On release: RadialMenu dismisses (`withTiming(0, {duration: 100})`), hit detection runs, entry logged if target captured

### Note Mode Persistence Indicator
- Note mode state persists across swipes (D-06)
- Glow ring remains visible on knob between swipes while note mode is active
- Ring must not obstruct hit targets or indicator dots (z-index below indicator dots, above knob fill)

### Note Mode Swipe Flow (UX-01)
- When note mode is active AND swipe completes without a hold: `pendingLogId` is set ŌåÆ NoteEntryModal opens
- NoteEntryModal appearance unchanged (existing implementation)
- When note mode is OFF: `pendingLogId` is never set on plain swipes
